# Weather App

A simple Python application that fetches real-time weather data using an API.

---

## Features
- Get current temperature
- Get humidity
- Get weather condition
- Get wind speed

---

## Technologies Used
- Python
- Requests library
- OpenWeather API

---

## How to Run

### 1. Install required package
pip install requests

### 2. Add your API key
api_key = "YOUR_API_KEY"

### 3. Run the program
python weather.py
